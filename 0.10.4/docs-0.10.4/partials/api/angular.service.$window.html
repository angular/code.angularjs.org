angular.service.$window

Description

A reference to the browser's window object. While window is globally available in JavaScript, it causes testability problems, because it is a global variable. In angular we always refer to it through the $window service, so it may be overriden, removed or mocked for testing.

All expressions are evaluated with respect to current scope so they don't suffer from window globality.

Example

  <input ng:init="$window = $service('$window'); greeting='Hello World!'" type="text" ng:model="greeting" />
  <button ng:click="$window.alert(greeting)">ALERT</button>
